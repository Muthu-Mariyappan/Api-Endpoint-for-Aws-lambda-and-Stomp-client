## Api-Endpoint-for-Aws-lambda-and-Stomp-client

## Requirements

1. Java - 1.8.x

2. Maven - 3.x.x

## Steps to Setup

**1. Clone the application**

```bash
git clone https://github.com/Muthu-Mariyappan/Api-Endpoint-for-Aws-lambda-and-Stomp-client.git
```

**2. Run the app with**

```bash
mvn spring-boot:run
```
